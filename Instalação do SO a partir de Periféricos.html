Instalação do SO a partir de periféricos

1-Escolha do Periférico

.Use uma unidade flash USB ou um DVD para criar um meio de instalação inicializável.

2-Obtenção do Sistema Operacional:

.Baixe a imagem ISO do sistema operacional desejado.

3-Formatação do Periférico:

.Formate a unidade flash USB ou insira o DVD.

4-Criação de uma Unidade Inicializável:

.Use ferramentas como Rufus (Windows) ou Etcher (Mac/Linux) para criar uma unidade inicializável.

5-Configuração da Inicialização no Computador:

.Acesse a configuração da BIOS/UEFI do computador.

.Configure a ordem de inicialização para USB/DVD.

6-Inicie o Computador a partir do Periférico:

.Reinicie o computador com a unidade flash USB ou DVD.

7-Siga o Processo de Instalação:

.Escolha idioma, fuso horário, etc.

.Selecione o disco para a instalação.

8-Conclusão da Instalação:

.Após a instalação, reinicie o computador.

.Remova o periférico inicializável.
